# Builder

Pre-processor for R (only works on POSIX systems).

## Install

Install from github with make.

```bash
git clone https://github.com/devOpifex/builder.git && make
```

## Usage

See `-help` for the complete list of options.

```bash
builder -input srcr -output R
```
